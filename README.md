# 🚀 AI-Powered Blockchain Analytics & Smart Dashboard Platform

## 📌 Overview
A full-stack AI + Web3 platform that provides real-time business analytics, predictive insights, and blockchain integration using Solana.

## 🔥 Features
- 🔐 Secure Admin Authentication (JWT-based)
- 📊 Real-time Analytics Dashboard
- 🤖 AI Sales Forecasting & Insights
- 🔗 Solana Wallet Integration (Phantom)
- 📁 CSV Data Upload & Processing
- ⚡ Fast API with Node.js & Express

## 🛠 Tech Stack
- Frontend: React (Vite), Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB
- AI/ML: Python / Custom Models
- Blockchain: Solana Web3.js
